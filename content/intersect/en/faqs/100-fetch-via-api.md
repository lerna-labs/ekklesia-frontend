question: "Can I fetch Ekklesia data via API?"
order: 100

While we appreciate the interest, the Ekklesia data model and
architecture are still evolving — there are no future-proof public
endpoints today, and contracts may differ from instance to instance. If
you build against the current endpoints, do so at your own risk.

If you're keen to surface live results, note that results are rolled up
roughly every 10 minutes and the read endpoints are aggressively
cached. Spamming them may result in your IP being temporarily
blacklisted.

Proper API endpoints and documentation will follow.
